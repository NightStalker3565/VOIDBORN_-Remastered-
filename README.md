Ohhh, *this thing?* Don’t worry about it. It’s just a teeny tiny harmless little game. A cute lil digital snack. Totally normal. Probably.

So like—yeah, video games these days? Basically anyone can pick one up. Tiny humans, big humans, sentient potatoes, you name it. But a lot of games feel like they’re made *for* younger kids, right? Bright colors, happy music, nothing weird lurking in the corners… 👀

So I thought: what if we took that idea… and gently tossed it into a mysterious, slightly ominous void?

✨ *Enter: VOIDBORN!* ✨

It’s a *lovely* lil game (and by “lovely” I mean “politely unsettling”) set in an alternate universe where everything is *almost* familiar… except Microsoft is called **Macrohard**, which is definitely fine and not suspicious at all.

In this world, you’ll be poking around using **MH-DOS**, a totally friendly command-line system that absolutely does not hide secrets it probably shouldn’t. You’ll explore the remains of **Reborn Laboratories**, a place that *definitely* followed all ethical guidelines and has *nothing* to hide. Nope. Nothing weird happened there. Don’t even worry about it.

As you dig deeper, you’ll start uncovering little crumbs of truth—logs, files, whispers of something that went very, very wrong. And the more you learn, the more that creeping feeling sets in…

You know the one.

That “oh no” feeling.

That “I am not alone” feeling.

That “something is maybe possibly definitely watching me through the screen right now but it’s fine haha” feeling.

Your goal?
Figure out what happened at Reborn Laboratories. Piece together the story. Expose the truth.

…before whatever *else* is in there decides you’ve seen enough.

---

🎮 **How to run this game:**

Just pop open your terminal and go:

```
pnpm install
pnpm dev
```

Then click the link it gives you, and boom—you’re in!
